### Earthquake Trends Analysis 🌏

An analysis and interpretation of global earthquake data to identify seismic patterns, trends and risk zones.

---------------------------------------------------------------------------------------------------------------------------------------------------------------
### ⚡ Project Overview

Earthquakes are one of the most destructive natural disasters. This project leverages the seismic data from the past 5 years to identify the trends, high risk zones and also provides an insight into the earthquakes that occured and the damages it caused during this period.

---------------------------------------------------------------------------------------------------------------------------------------------------------------
### ✔ Key Objectives

📍 Helps in analyzing the frequency and magnitude changes ovrertime.

📍 Enables governments, insurers and researchers to assess earthquake risks, plan disaster management strategies which supports in informed decision-making process.

📍 Helps to forecast future activities using Machine Learning.

---------------------------------------------------------------------------------------------------------------------------------------------------------------
### 🛠 Methods & Features

📍 Retrival of earthquake data from the USGS API.

📍 Cleaning and preprocessing of the data using Python. Converting timestamps, handling missing values and derive new analytical columns such as year, month etc., for consistency and accuracy.

📍 Store the cleaned and structured dataset with all features in MySQL and perform detailed analytical exploration, uncovering trends and insights in global seismic activity.

📍 Establishment of an interactive Streamlit dashboard

---------------------------------------------------------------------------------------------------------------------------------------------------------------
### 📌 Conclusion

Through the real time data retrival and the implementation of interactive visualisation, this project transforms raw data into actionable insights. the ultimate value of this project is its ability to support informed decision making. By identifying earthquake frequency patterns and high risk zones over the past 5 years this helps in forecasting future seismic threats and optimizing disaster management strategies.
